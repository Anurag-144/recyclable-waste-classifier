# Recyclable Waste Classifier

ML model to classify recyclable materials into 6 categories.

## Categories
- Cardboard
- Glass  
- Metal
- Paper
- Plastic
- Trash

## Tech Stack
- Python 3.12
- TensorFlow/Keras
- MobileNetV2 (Transfer Learning)
- Streamlit (Demo)

## Results
- Training Accuracy: 89%
- Validation Accuracy: 79%

## Run Demo
streamlit run week4.py
